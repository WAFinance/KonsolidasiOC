     Petunjuk Pengisian

    1. Pastikan dokumen telah ditandatangani dan distempel perusahaan.
    2. Format file: PDF, ZIP, atau RAR (maks. 25 MB).
    3. Isi semua kolom bertanda (*) dengan lengkap dan benar.
    4. Periksa kembali nama perusahaan dan NPWP sebelum mengirim.
    5. Upload hanya dilakukan sekali; untuk revisi hubungi Pokja Pemilihan.

    URL deployment Google Apps Script. Ini adalah kunci agar semua pengaturan tersinkron global.

    Penting: Semua pengaturan admin (judul, jadwal, dokumen, dll) tersimpan di Google Spreadsheet via Apps Script dan berlaku untuk semua pengguna di semua browser/perangkat. Pastikan URL sudah dikonfigurasi.
    Langkah deploy:
    1. Buka Spreadsheet → Extensions → Apps Script
    2. Paste kode dari file code.gs
    3. Deploy → New deployment → Web App → Execute as: Me, Who has access: Anyone
    4. Salin URL deployment ke kolom di atas
