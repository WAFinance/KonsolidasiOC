Unggah dokumen penawaran Anda melalui formulir di bawah ini.

 Petunjuk Pengisian

  1. Pastikan dokumen penawaran telah ditandatangani dan distempel perusahaan.
  2. Format file yang diterima: PDF, ZIP, atau RAR (maksimal 25 MB).
  3. Isi semua kolom yang bertanda (*) dengan lengkap dan benar.
  4. Pilih file dokumen dari komputer Anda pada kolom Upload Dokumen.
  5. Periksa kembali nama perusahaan dan nomor NPWP sebelum mengirim.
  6. Setelah berhasil submit, simpan bukti/tanda terima untuk arsip.
  7. Upload hanya dilakukan sekali; untuk revisi hubungi Pokja Pemilihan.
Pilih file dari komputer Anda. File akan diunggah ke Google Drive secara otomatis.
